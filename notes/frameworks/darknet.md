# Darknet

Installation

- `git clone https://github.com/AlexeyAB/darknet darknet_ab` 
- if `make` errors
	- `export LD_LIBRARY_PATH=/usr/local/cuda-9.0/lib64:$LD_LIBRARY_PATH`
	- `export PATH=/usr/local/cuda-9.0/bin:$PATH`
    - `make clean && make`
